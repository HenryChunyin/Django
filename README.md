# Django LIMS Demo

A practice Django project for building a simple laboratory information management interface.

## Purpose

This project was developed as a personal practice project to understand how web applications can support laboratory workflow management, including sample records, data entry forms, and database-backed pages.

## Tech Stack

- Python
- Django
- SQL concept
- HTML templates
- Basic database models and views

## Features

- Sample / record management pages
- Form-based data entry
- Database-backed workflow structure
- Basic admin interface

## Status

This is a learning and demonstration project, not a production system.
